# open-odyssey

Odyssey beta public feedback and issue tracking.

## What is Odyssey?

Odyssey is a local-first, Obsidian-powered personal memory system. It remembers across conversations — facts, corrections, references — and stores everything as readable Markdown in your own vault. The Obsidian plugin is forever free.

## Beta Feedback

Found a bug, a memory glitch, or have a feature idea? Open an issue here.

**Privacy reminder**: never paste private memory contents, API keys, tokens, raw `.odyssey/` Shadow Index files, or unredacted vault paths into public issues. Paraphrase is enough.
